# HTML Jenkins Demo

A demonstration project showcasing HTML development with Jenkins CI/CD integration.

## Technologies
- HTML (100%)

## About
This repository contains a simple HTML project demonstrating continuous integration and deployment patterns using Jenkins.

## Getting Started
1. Clone the repository
2. Set up Jenkins CI/CD pipeline
3. Configure webhooks to trigger builds on push
4. Deploy to your web server

## Features
- Simple HTML web application
- Jenkins pipeline integration
- Automated build and deployment
- CI/CD demonstration
